在hexo中输入中文

2016-08-04
  • hexo

在命令行直接输入hexo new 中文名称时，生成的md文件会出现乱码，即使在_config.yml修改语言也没有效果。目前没找到完美的解决方法，可替换的选择是hexo new一个英文名称，然后再修改title。
